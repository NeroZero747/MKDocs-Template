# Welcome to Pro Docs

This starter kit gives you a professional MkDocs setup with:

- **Grids and Cards** for clean layouts
- **Buttons** and **Admonitions**
- **Expanders/Details** blocks
- **Beautiful Tables** (striped/hover/compact)
- **Mermaid diagrams** synced with light/dark mode
- **Subtle transitions** and a **Bootstrap-ish** feel

> Tip: Use the theme switcher (top-right) to see colors change everywhere—tables, cards, buttons, and Mermaid.
